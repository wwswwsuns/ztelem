## ⚠️ 注意事项

### 1. **兼容性**
- 新的扩展版本与原版本在接口上基本兼容
- 配置文件格式有扩展，但向后兼容

### 2. **内存使用**
- 扩展版本会使用更多内存（缓冲区更大）
- 建议监控内存使用情况

### 3. **数据库连接**
- 新版本使用更多数据库连接
- 确保数据库服务器支持足够的连接数

### 4. **日志级别**
- 生产环境建议使用 `info` 或 `warn` 级别
- 避免使用 `debug` 级别以减少日志量
