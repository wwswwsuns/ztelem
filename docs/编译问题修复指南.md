## 🔧 遇到的问题及解决方案

### 1. **类型重复声明错误**
```
Config redeclared in this block
DatabaseConfig redeclared in this block
```

**原因**: 同时存在 `config.go` 和 `config_original.go` 文件，导致类型重复声明。

**解决方案**:
```bash
rm internal/config/config_original.go
rm internal/database/database_original.go
rm internal/buffer/buffer_manager_original.go
```

### 2. **类型名称不匹配错误**
```
undefined: database.DB
undefined: buffer.BufferManager
```

**原因**: 扩展版本中类型名称发生了变化。

**解决方案**: 修改 `internal/collector/simple_collector.go`
```go
// 修改前
db            *database.DB
bufferManager *buffer.BufferManager

// 修改后
db            *database.ExtendedDB
bufferManager *buffer.ExtendedBufferManager
```

### 3. **函数名称不匹配错误**
```
undefined: buffer.DefaultBufferConfig
undefined: buffer.NewBufferManager
```

**解决方案**: 更新函数调用
```go
// 修改前
bufferConfig := buffer.DefaultBufferConfig()
bufferManager := buffer.NewBufferManager(bufferConfig, logger, db)

// 修改后
bufferConfig := config.BufferConfig{...}
writerConfig := config.DatabaseWriterConfig{...}
bufferManager := buffer.NewExtendedBufferManager(bufferConfig, writerConfig, logger, db)
```

### 4. **main函数重复声明**
```
main redeclared in this block
```

**原因**: 存在多个main文件。

**解决方案**:
```bash
rm main_backup.go main_original.go
```

### 5. **未定义的函数错误**
```
undefined: collector.NewExtendedCollector
```

**解决方案**: 使用正确的函数名
```go
// 修改前
telemetryCollector := collector.NewExtendedCollector(...)

// 修改后
telemetryCollector := collector.NewSimpleCollector(log, db)
```

### 6. **缺少导入包**
```
undefined: sql
```

**解决方案**: 添加导入
```go
import (
    "database/sql"
    // 其他导入...
)
```

### 7. **未使用的导入**
```
"telemetry-collector/internal/logger" imported and not used
```

**解决方案**: 删除未使用的导入

### 8. **函数不存在错误**
```
undefined: runtime.SetGCPercent
```

**解决方案**: 使用正确的包
```go
// 修改前
runtime.SetGCPercent(perfConfig.GCPercent)

// 修改后
debug.SetGCPercent(perfConfig.GCPercent)
```

### 9. **变量名冲突**
```
debug already declared through import of package debug
```

**解决方案**: 重命名变量
```go
// 修改前
debug = flag.Bool("debug", false, "启用调试模式")

// 修改后
debugMode = flag.Bool("debug", false, "启用调试模式")
```
